# Quick Evaluation Checklist

Use this checklist while showing the demo.

| PDF requirement | Where to show it in project/demo |
|---|---|
| Resume PDF input | Upload resume on first screen |
| Resume parsing | Show candidate summary and extracted skills |
| Role inference | Show recommended roles and match scores |
| Interview focus areas | Show focus areas/weak areas from resume analysis |
| Job recommendations | Open job recommendations section |
| Dynamic questions | Start interview and show generated question |
| Adaptation | Give weak answer, then show easier follow-up |
| Speech-to-text | Use speech mode and speak an answer |
| Audio intelligence | Show confidence, pauses, fillers, clarity |
| Visual intelligence | Turn on webcam and show engagement/eye contact |
| Technical evaluation | Submit answer and show correctness/depth scores |
| Multimodal scoring | Show combined technical + audio + video feedback |
| Explainable feedback | Show final feedback report and practice plan |
| README/setup | Show `README.md` |
| Architecture document | Show `docs/architecture.md` |
| Sample input/output | Show files inside `samples/` |
